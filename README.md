# Travel Express

Travel Express is a modern travel website designed to showcase destinations, facilities, and booking options for travelers.  
It provides a clean UI, responsive design, and interactive features like contact forms, booking modals, and integrated maps.

---

## Features
-  **Main Page**: Highlights destinations, advantages, and cultural experiences.  
-  **About Page**: Information about travel areas and services offered.  
-  **Content Page**: Professional details with booking options, flight info, visa assistance, and tour guide facilities.  
-  **Contact Page**: Office details, contact form, and Google Maps integration.  
-  **UI Design**: Built with HTML, CSS (Grid + Flexbox), and aesthetic color combinations.  
-  **Interactivity**: JavaScript for form validation, booking popups, and dynamic content.  

---

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript  
- **Design**: Responsive layout using Grid & Flexbox  
- **Integration**: Google Maps Embed for location
